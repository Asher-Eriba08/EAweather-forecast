# EAweather-forecast
A simple weather web application built using Python and Flask.
It allows users to enter a city name and get real-time weather information along with the local time.

🚀 Features
🌍 Search weather by city name
🌡️ Displays temperature in °C
💧 Shows humidity levels
🌬️ Wind speed information
🕒 Displays local time of the city
❌ Error handling for invalid input
 Built With;
Python
Flask
Requests
HTML (Jinja Templates)
📁 Project Structure
project/
│
├── app.py
├── requirements.txt
├── templates/
│   └── index.html
└── .gitignore
⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/Asher-Eriba08/EAweather-forecast
Navigate into the project folder:
cd EAweather-forecast
Install dependencies:
pip install -r requirements.txt
Run the app:
python app.py
Open in browser:
http://127.0.0.1:5000
🔑 API Used
Weather data: https://wttr.in
Time data: TimeZoneDB API
This project was built as a beginner-friendly Flask application to practice working with APIs and backend development.
